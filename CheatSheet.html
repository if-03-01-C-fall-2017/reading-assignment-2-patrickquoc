HTML

Basics and Elements

HTML files always starts with <​!DOCTYPE html>. The document itself starts with <​html>. The visible part of the file is the <​body>.
Paragraphs are made with <​p> and links with <​a>. You can display preformatted text with <​pre>.
Headers are made with <​h1> which is in this case the largest header. For smaller header just change the number to a greater number than 1.
You can put images in your document with <​img>. For example: <​img src="example.jpg" width="200" height="200">
The picture will look like this.

Attributes

All HTML elements can have attributes. Attributes provide additional information about a element.
Attributes are always specified in the start tag.
Attributes usually come in name/value pairs like: name="value"

Here are some important attributes:

href: The link address in a <​a> element is specified in the href attribute: <​a href="https://www.Example.com">This is a link<​/a>
src: The filename of the image source is specified in the src attribute: <​img src="example.jpg">
width/height: These elements sets the size of the image: <​img src="example.jpg" width="200" height="200">
alt: The alt attribute specifies an alternative text to be used, when an image cannot be displayed: <​img src="example.jpg" alt="This is an example">
style: The style attribute is used to specify the styling of an element, like color, font, size etc.: <​p style="color:red">I am a paragraph<​/p>
lang: This declares the language. This is important for accessibility application and search engines.: <​html lang="en-US">
title: The value of the title attribute will be displayed as a tooltip when you mouse over the paragraph: <​p title="I'm a tooltip">

Comments

You can use comments to describe your code, for notifications and reminders and it is a great way for debugging a code. Comments are not displayed in the browser.
Example: <​!-- Your Comment -->
There is also conditional comments but it can be only executed by Internet Explorer.
Example:
<​!--[if IE 9]>
.... some HTML here ....
<​![endif]-->

Links

Links are made with the <​a> element.
The href attribute specifies the destination address of the link. It can be a full web address or a local link.
The link text is the visible part of the element. The target attribute specifies where to open the linked document. The target attribute can have one of the following values:
_blank - Opens the linked document in a new window or tab
_self - Opens the linked document in the same window/tab as it was clicked (this is default)
_parent - Opens the linked document in the parent frame
_top - Opens the linked document in the full body of the window
framename - Opens the linked document in a named frame
Example: <​a href="https://www.example.com/" target="_blank">This opens a new tab with the website.<​/a>

Images

The style attribute is important for preventing internal or external style sheets from changing the original size of images. For Example: <​style> img {width:100%;}<​/style>
You can find images in another folder by giving the correct path of the images: <​img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
If you want to take a picture from the internet, just put your link in src.
You can also use animated pictures like gifs on your site. Just pick a .gif file and your good to go.

Head

The <​head> element is a container for metadata (data about data) and is placed between the <​html> tag and the <​body> tag.
HTML metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, links, scripts, and other meta information.
The following tags describe metadata: <​​title>, <​​style>, <​​meta>, <​​link>, <​​script>, and <​​base>.
The <​​title> element defines the title of the document. It defines the title of the browser tab, provides a title when added to the favourites and displays the title in the search engine.
The <​​meta> element is used to specify which character set is used, page description, keywords, author, and other metadata.

CSS

Syntax

A CSS rule-set consists of a selector and a declaration block. First comes the selector then the declaration.
Inside the declaration comes the property then the value.

Selectors

CSS selectors are used to select HTML elements based on their element name, id, class, attribute, and more.
If you have elements with same style definitions you can group selectors by seperating the elements with a comma.

Comments

Comments helps the coder to understand the code at a later date. Comments are done like this /* Example text */

Colors

HTML colors can be specified using predefined color names, RGB, HEX, HSL, RGBA, and HSLA values.
There are 140 standard color names in HTML. (e.g. Tomato, Orange, Violet,...)
In HTML, a color can be specified using a hexadecimal value in the form: #rrggbb The highest value (for example red) is ff.

Background

The background-color property specifies the background color of an element.
You can change the background repeat vertically/horizontally by using background-repeat: repeat-x;. If you don't like to repeat it use no-repeat instead of repeat-x.
If you want the background fixed on a position use background-attachment: fixed;.

Text

The color property is used to set the color of the text. You are able to do this on links too.
The text-align property is used to set the horizontal alignment of a text.
The text-decoration property is used to set(e.g. underline) or remove decorations from text.
The text-transform property is used to specify uppercase and lowercase letters in a text.

Fonts

Serif fonts have small lines at the ends on some characters, which Sans-Serif ("Sans" means without) doesn't have these lines.
The font family of a text is set with the font-family property. Example: p { font-family: "Times New Roman", Times, serif; }
You can set the font size pixels(font-size=40px;) or Em(font-size=2.5em;).
The font-weight property specifies the weight of a font: p.thick{font-weight: bold}

Combinators

Between selectors, we can include a combinator. There are 4 diffrent combinators in CSS3: descendant(space), child(>), adjacant sibling(+) and general sibling(~) selector.

Pseudo Classes

A pseudo-class is used to define a special state of an element. It can be used to different unvisited and visited links, style an element when a mouse hovers over it or when it gets on focus.
The syntax looks like this: selector:pseudo-class { property:value; }
The :first-child pseudo-class matches a specified element that is the first child of another element.
